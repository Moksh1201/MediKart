🩺 MediCart – Smart Medicine & Health Management System
MediCart is a full-stack web application designed to simplify the process of buying medicines, booking medical tests, and managing prescriptions, while also providing robust administrative and medical oversight.

🔍 Key Features
✅ Buy Medicines Online

Users can browse and purchase a wide range of medicines.

Some medicines require a valid prescription, verified by a registered doctor before approval.

✅ Prescription-Based Orders

Critical medicines are locked behind prescription checks.

Doctors review uploaded prescriptions before the order is confirmed.

✅ Book Appointments for Blood Tests

Users can book blood test appointments directly from the platform.

Lab partners and test availability are synced with the admin dashboard.

✅ Doctor Verification Panel

Registered doctors can log in and verify prescriptions.

Doctors have dashboards showing patients, approvals, and prescription history.

✅ Admin Panel

Manage all aspects: users, doctors, prescriptions, test bookings, and medicine inventory.

Track stock levels, restock alerts, sales data, and prescription validation reports.

✅ Medicine Inventory with FS Storage

Medicines are stored in a JSON database (via Node.js & fs module).

Each medicine includes name, company, usage, category, expiry date, stock count, and prescription requirement.

✅ Notifications & Alerts

Low stock alerts for medicines.

Confirmation and alerts for prescription approvals and test bookings.

👥 Roles
User: Buy medicines, upload prescriptions, book lab tests.

Doctor: Validate prescriptions, approve critical medicine orders.

Admin: Manage everything – doctors, users, inventory, orders, and more.
